Que es MikeComputers

MikeComputers es un proyecto creado para el curso Java full stack de EOI.
Esta web esta dedicada a informar sobre las principales noticias de videojuegos, informática y smartphones.
En ella, podremos encontrar información útil para los usuarios interesados en el mundo de la tecnología.

MikeComputers es una web aún en construcción, por lo que es posible que tenga pequeños defectos, pero seguiré trabajando para convertirla en una web de referencia.

Esta web ha sido creada exclusivamente por Michael López Martínez, tanto el lado backend como la parte frontend.
Pare ello, he usando herramientas como Ionic, Angular y Spring.
